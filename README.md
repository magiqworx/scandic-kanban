# Kanban MVP

A minimal, single-board Kanban application built with Next.js, featuring a clean Scandinavian aesthetic and Bauhaus typography.

## Features
- Single board with 5 fixed, renamable columns
- Drag-and-drop interface for task management
- Lightweight card creation and deletion
- Pre-populated dummy data on initialization
- Zero persistence or user management for maximum simplicity

## Tech Stack
- Next.js (Client-rendered)
- Tailwind CSS (Black & White color scheme)

## Development

To run the application locally:

1. Navigate to the frontend directory:
   cd frontend

2. Install dependencies:
   npm install

3. Start the development server:
   npm run dev

Open http://localhost:3000 to view the application.
